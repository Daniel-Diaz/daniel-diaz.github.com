data ImpulseSeries key isp st en rep Source

'ImpulseSeries is the fundamental datatype that represents all the various kinds of time related event data.

It could be a-periodic sampling from a cell modem, a set of compressed files etc.. .

The idea is any set of values represented by 'rep is expected to have period 'isp, which may be a parameterized value. The series start value ('st) and end value ('en) are there to be used to identify individual data points.

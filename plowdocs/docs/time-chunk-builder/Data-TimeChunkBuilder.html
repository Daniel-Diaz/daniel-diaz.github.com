Builder operations

-> IO (TimeChunkState e, Either (TimeChunkEnv e) (Vector (Maybe e))) 

Build a time chunk using the TimeChunkBuilder monad. The chunk is returned as a vector with potentially empty values (Nothing). If the chunk building is unfinished, it returns an environment that you can use to resume the chunk building using buildChunkFromEnv. In addition, values inserted with the chunk builder with time in the future or the past are returned in a TimeChunkState value.

Resuming buildings

data TimeChunkEnv e Source

A Time Chunk Builder environment. You can use a value of this type to resume building a chunk with buildChunkFromEnv. The type is annotated with the type of the values in the chunk.

buildChunkFromEnv :: TimeChunkEnv e -> TimeChunkBuilder e () -> IO (TimeChunkState e, Maybe (Vector (Maybe e))) Source

Resume a chunk building using a supplied environment. As in buildChunk, the building might not finish. If this is the case, it returns Nothing instead of the vector. The chunk building can be continued using the same environment again.

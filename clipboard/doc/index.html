System clipboard interface with unicode support.

Clipboard interface

setClipboardString :: String -> IO ()

Puts a string on the clipboard, using the cF_UNICODETEXT clipboard format.

getClipboardString :: IO (Maybe String)

Gets the contents of the clipboard as a String. Returns Nothing if the clipboard content is not of type cF_TEXT or cF_UNICODETEXT, i.e. it doesn't contain textual data.

modifyClipboardString :: (String -> String) -> IO Bool

Modifies the clipboard content. If the clipboard has textual data, this function modifies its content, and return True. Otherwise, it does nothing and return False.

Clipboard format

cF_UNICODETEXT :: ClipboardFormat

Clipboard format of Unicode text.
